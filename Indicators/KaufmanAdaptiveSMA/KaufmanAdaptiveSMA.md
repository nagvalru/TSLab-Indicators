# KaufmanAdaptiveSMA

## 1. Название индикатора

- Имя: `KaufmanAdaptiveSMA`
- Короткое пояснение: адаптивная простая средняя на переменном периоде

## 2. Назначение

Индикатор считает простую среднюю ряда `source`, но длина окна берётся не из одного числа, а из входного ряда `varPeriod`.

Он нужен в тех стратегиях, где средняя должна менять свой горизонт вместе с текущим состоянием рынка.

## 3. Проект и стратегия первого использования

- Проект первого использования: `KaufmanStrategies`
- Стратегия первого использования: `TSM Dynamic Breakout-Volatility System`

## 4. Тип индикатора

Это адаптивный расчётный индикатор с периодом-серией.

## 5. Логика расчёта

На каждом баре индикатор:

1. читает текущее значение `varPeriod`
2. округляет его до целого
3. ограничивает минимумом `1`
4. берёт соответствующее окно последних значений ряда `source`
5. возвращает обычное арифметическое среднее на этом окне

Формально:

`result[i] = sum(source[j]) / period`, где `j` пробегает текущее адаптивное окно

## 6. Входы индикатора

- `source`
  - тип: числовой ряд
  - смысл: ряд, который усредняется
- `varPeriod`
  - тип: числовой ряд
  - смысл: адаптивная длина окна
  - ограничение: итоговый период не меньше `1`

## 7. Выходы индикатора

Один числовой ряд: адаптивная простая средняя.

## 8. Параметры пользователя

Прямых скалярных параметров нет. Весь контроль идёт через входной ряд `varPeriod`.

## 9. Применение в стратегиях

В `TSM Dynamic Breakout-Volatility System` индикатор используется в альтернативной ветке расчёта, когда разности `ediff` и `xdiff` строятся не через обращение к исторической цене, а через адаптивную среднюю:

- `adaptiveSmaVarA`
- `adaptiveSmaVarB`

Это позволяет переключить трактовку стратегии без создания ещё одного сложного блока формулы.

## 10. Ограничения и особенности реализации

- Период округляется до целого.
- Минимум периода равен `1`.
- На ранней истории используется доступная часть окна.
- Индикатор не содержит внутренней оптимизации и зависит от качества ряда `varPeriod`.

## 11. Проверка корректности

Проверочный сценарий:

- подать постоянный `varPeriod`, например `20`
- сравнить выход индикатора со стандартной `SMA(20)`

При совпадении рядов базовая реализация подтверждается.

## 12. Неоднозначности

Индикатор детерминирован. Все неоднозначности относятся только к выбору стратегии: когда именно использовать ветку со средней вместо исторического сдвига.
