# KaufmanAdaptiveLowest

## 1. Название индикатора

- Имя: `KaufmanAdaptiveLowest`
- Короткое пояснение: адаптивный минимум на переменном периоде

## 2. Назначение

Индикатор считает минимум ряда `source` на окне переменной длины. Длина окна на каждом баре задаётся рядом `varPeriod`.

Он нужен для тех стратегий, где нижняя граница канала или защитный рубеж должны меняться вместе с адаптивным периодом.

## 3. Проект и стратегия первого использования

- Проект первого использования: `KaufmanStrategies`
- Стратегия первого использования: `TSM Dynamic Breakout-Volatility System`

## 4. Тип индикатора

Это расчётный адаптивный индикатор с периодом-серией.

## 5. Логика расчёта

На каждом баре индикатор:

1. берёт текущее значение `varPeriod`
2. округляет его до целого
3. ограничивает минимумом `1`
4. просматривает последние `period` значений ряда `source`
5. возвращает минимум на найденном окне

Формально:

`result[i] = min(source[j])`, где `j` пробегает окно от `i - period + 1` до `i`

## 6. Входы индикатора

- `source`
  - тип: числовой ряд
  - смысл: ряд, по которому ищется минимум
- `varPeriod`
  - тип: числовой ряд
  - смысл: адаптивная длина окна
  - ограничение: фактическое окно не меньше `1`

## 7. Выходы индикатора

Индикатор возвращает один числовой ряд: минимум исходного ряда на адаптивном окне.

## 8. Параметры пользователя

Собственных скалярных параметров нет. Пользователь управляет индикатором через ряд `varPeriod`, который формируется в стратегии.

## 9. Применение в стратегиях

В `TSM Dynamic Breakout-Volatility System` индикатор используется для расчёта адаптивных нижних рубежей:

- `lowestAdjEntry`
- `lowestLow`
- `lowestAdjExit`
- `lowestLowB`

Это даёт стратегии возможность строить стопы и пробойные уровни не на фиксированном окне, а на текущем рабочем масштабе рынка.

## 10. Ограничения и особенности реализации

- Период округляется до целого.
- Значение периода меньше `1` автоматически заменяется на `1`.
- В начале истории используется доступная длина окна.
- Если `varPeriod` ведёт себя неустойчиво, индикатор будет честно повторять эту неустойчивость.

## 11. Проверка корректности

Для проверки можно подать фиксированный период, например `15`, и сравнить результат с обычным `Lowest(source, 15)`.

## 12. Неоднозначности

Неоднозначностей в самом индикаторе нет. Они возможны только в логике формирования `varPeriod` на стороне стратегии.
