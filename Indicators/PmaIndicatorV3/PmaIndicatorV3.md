# PmaIndicatorV3

## Назначение

`PmaIndicatorV3` рассчитывает Prosvirin Moving Average: значение линии линейной регрессии, спроецированное на последний бар каждого скользящего окна.

## Parameters

- `Period` - period of the linear regression window.

## Параметры

- `Period` - период окна линейной регрессии.

## Inputs

- `Source` - input numeric series.

## Входы

- `Source` - входной числовой ряд.

## Output

- Numeric series with PMA values.

## Выход

- Числовой ряд со значениями PMA.
